# Mon Année Grand Frais — Wrapped 2026

Prototype interactif hébergé sur GitHub Pages.

👉 https://alexandrefanonniji.github.io/roadmapgrandfrais/

## Structure

- `index.html` — point d'entrée (charge React + Babel, puis les fichiers JSX)
- `app.jsx`, `primitives.jsx`, `user-data.jsx`, `share-overlay.jsx` — logique React
- `screens/` — les 9 écrans du Wrapped
- `styles.css`, `fonts/`, `icons/` — assets visuels
- `.nojekyll` — désactive le traitement Jekyll de GitHub Pages
